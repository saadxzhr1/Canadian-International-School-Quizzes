Space Pilot
Events Keyboard Input Motion

Goal: Control Rocketship in all four directions using the arrow keys — fly past Bat and Bee to reach the Star!

►

🎭 Sprites & Starting Positions

🚀 Rocketship Center: x:0 y:0 Visible, size 80
🦇 Bat Left side: x:-80 y:50 Visible, size 60
🐝 Bee Top: x:60 y:100 Visible, size 60
⭐ Star Far right: x:170 y:-80 Visible, size 80

ℹ️ "Rocketship" is in the Space category of the Scratch sprite library. Bat and Bee are obstacles — they don't move, just block the path.

📋 Steps

  1. 1Click Rocketship. From Events, drag when [right arrow] key pressed onto the scripts area. From Motion, attach change x by (10) below it.
  2. 2Drag a new when [left arrow] key pressed block. Attach change x by (-10). Moving left means negative x.
  3. 3Drag when [up arrow] key pressed. Attach change y by (10).
  4. 4Drag when [down arrow] key pressed. Attach change y by (-10). You now have 4 separate scripts — one per arrow key.
  5. 5Click the green flag. Use the arrow keys to fly Rocketship past Bat and Bee to reach the Star in the bottom-right!
⭐ Challenge

Change all four 10 values to 20 for turbo speed. Then add a 5th script: when [space] key pressed → go to x:(0) y:(0) — space bar teleports Rocketship back to the center.

2
The Glowing Deep
Forever Loop Bounce If Touching

Goal: Jellyfish bounces and glows when it touches Fish — Fish flips direction when it touches Crab!

🪼
🐟
🦀
⭐

🎭 Sprites & Starting Positions

🪼 Jellyfish Center: x:0 y:30 Visible, size 100
🐟 Fish Right side: x:100 y:-30 Visible, size 80
🦀 Crab Bottom-left: x:-120 y:-130 Visible, size 70
⭐ Starfish Top-right: x:150 y:120 Visible, size 60

ℹ️ if <touching [Sprite]> then is in Control + Sensing. Drag the touching oval from Sensing into the if slot. Starfish is decorative — no script needed.

📋 Steps

  1. 1Click Jellyfish. Add when 🏴 clicked → set rotation style [left-right] from Motion. Then attach a forever loop from Control.
  2. 2Inside Jellyfish's forever: add move (5) steps + if on edge, bounce. Then add if <touching [Fish]> then → change [color] effect by (25) from Looks. Jellyfish glows every time it hits Fish!
  3. 3Click Fish. Add when 🏴 clicked → set rotation style [left-right] → forever. Inside: move (3) steps + if on edge, bounce.
  4. 4Still inside Fish's forever: add if <touching [Crab]> then → turn (180) degrees from Motion. Fish flips direction each time it hits Crab!
  5. 5Click Crab. Add when 🏴 clicked → set rotation style [left-right] → forever → move (1) steps + if on edge, bounce. Crab barely creeps. Click the green flag!
⭐ Challenge

Add a second if <touching [Jellyfish]> then inside Fish's forever: make Fish also change [color] effect by (20) when Jellyfish hits it — the glow goes both ways. Then change Crab's speed from 1 to 3 and see how it changes the chaos.

Dodge, Grab & Run!
Random Position If Touching Say Block

Goal: Apple spawns at a random spot — grab it while dodging the Robot — then race to the Star and shout "Made it!!"

🤖
🍎
🧒
⭐

🎭 Sprites & Starting Positions

🧒 Avery Top-right: x:170 y:120 Visible, size 80
🤖 Robot Center: x:0 y:0 Visible, size 100
🍎 Apple Random — set by script Visible, size 70
⭐ Star Bottom-left: x:-170 y:-110 Visible, size 80

ℹ️ go to (random position) is in Motion. hide and show are in Looks. say [ ] (2) secs is also in Looks.

📋 Steps

  1. 1Click Robot. Add when 🏴 clicked → set rotation style [left-right] → forever → move (5) steps + if on edge, bounce. Robot patrols left and right forever.
  2. 2Click Apple. Add when 🏴 clicked → show → go to (random position). Apple appears somewhere different every game! Then add a forever → if <touching [Avery]> then → hide. Apple disappears when Avery grabs it.
  3. 3Click Avery. Build the same 4 arrow-key scripts from Exercise 1 — right/left/up/down — each with change x/y by (15).
  4. 4Add a second when 🏴 clicked to Avery → forever → if <touching [Star]> then → say [Made it!!] (2) secs. Avery cheers when reaching the Star!
  5. 5Click the green flag. Apple spawns randomly — navigate Avery to grab it first, dodge Robot, then reach the Star at bottom-left. Every restart, Apple is somewhere new!
⭐ Challenge

Speed up Robot: change move (5) steps to move (9) steps. Then add a second forever on Robot: change [color] effect by (15) — it flashes as it hunts. Change Avery's say text to something you actually want to shout.
