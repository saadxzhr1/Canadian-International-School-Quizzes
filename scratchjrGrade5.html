1
The Heist 🏃
Sequences Messages Multiple Sprites

Goal: Thief grabs the gem and runs — Guard chases — Dog blocks the escape!

🎭 Sprites & Starting Positions

🧒 Thief Left edge, middle height Visible
👮 Guard Far left, below Thief Visible
🐕 Dog Right edge, off-screen Visible
⭐ Gem (Star) Center-right of scene Visible, normal size

📋 Steps

  1. 1Tap Thief. Add 🏴 Green Flag → Move right 5 × 3. At the end, add Send message 🔴. Thief runs toward the gem.
  2. 2Tap Gem. Add When I receive 🔴 → Hide. Gem disappears the moment Thief arrives.
  3. 3Tap Guard. Add When I receive 🔴 → Move right 4 × 4. Guard starts chasing right after the gem vanishes.
  4. 4Tap Dog. Add 🏴 Green Flag → Wait 2 → Move left 5 × 2 → Send message 🔵. Dog charges in from the right after 2 seconds.
  5. 5Tap Thief again. Add When I receive 🔵 → Jump → Jump. Thief is trapped between Guard and Dog — panic!
⭐ Challenge

After Thief's second jump, add Move left 4 × 2 — Thief tries to run back. Reposition Guard so they collide in the middle. Who reaches who first?

2
The Haunted House 🦇
Tap Interactions Scene Changes Hide

Goal: Tap the bat to trigger a jump-scare — then get warped to the next haunted room!

🎭 Sprites & Starting Positions

👦 Avery Left side, center height Visible, normal size
🐕 Dog Right edge, off-screen Visible, normal size
🐈 Cat Scene 2 — center Visible, normal size
🦇 Bat Center of Scene 1 Visible, normal size

⚠️ No ghost in ScratchJr — use Bat (Animals category) as the scary sprite. Cat and Dog belong in Scene 2 only.

📋 Steps

  1. 1Tap the + scene button (bottom of screen) to create Scene 2. Add Cat and Dog there. Go back to Scene 1 to continue.
  2. 2Tap Bat. Add 👆 When tapped → Grow 5 → Shrink 5 → Send message 🔴. Bat lunges forward when tapped.
  3. 3Tap Avery. Add When I receive 🔴 → Jump → Jump → Move left 3. Avery jumps twice and stumbles back.
  4. 4After Avery's moves, add Wait 1 → Change scene →. One second after the scare, everyone warps to Scene 2.
  5. 5In Scene 2, tap Cat. Add 🏴 Green Flag → Grow 4 → Shrink 4. Cat was waiting there all along. Tap the bat to start!
⭐ Challenge

In Scene 1, Dog is hidden off the right edge. Add When I receive 🔴 → Move left 7 × 2. Dog charges in from off-screen at the same moment Bat lunges. Two jump-scares at once.

3
The Ambush 😱
Say Block Messages Loops

Goal: Avery walks into a trap — a dinosaur leaps out — Avery jumps and shouts in fear!

🎭 Sprites & Starting Positions

👦 Avery Far left, center height Visible, normal size
🦕 Dino Right side, off-screen bottom Hidden at start
🐈 Cat Center-right, above Dino Visible, normal size
🐦 Bird Top-right corner Visible, normal size

ℹ️ The Say block (💬) is in the yellow Looks blocks — it shows a speech bubble above the sprite. Tap it to type your message.

📋 Steps

  1. 1Tap Dino. Since it starts hidden, first add 🏴 Green Flag → Hide to lock its starting state.
  2. 2Tap Avery. Add 🏴 Green Flag → Move right 4 × 4. At the end, add Send message 🔴. Avery walks toward the trap.
  3. 3Tap Dino. Add When I receive 🔴 → Show → Grow 4 → Send message 🔵. Dino appears and grows when Avery arrives.
  4. 4Tap Avery. Add When I receive 🔵 → Jump → Say "AHHH!!" → Jump → Say "HELP!!" → Move left 5. Avery shouts while jumping and runs!
  5. 5Tap Cat. Add When I receive 🔵 → Move right 3 → Say "haha". Cat was in on it. Tap the green flag!
⭐ Challenge

Add Bird on When I receive 🔴 → Repeat 3: inside put Move down 2 then Move up 2. Bird swoops nervously as Dino appears. Change Avery's Say text to something funnier — it's your choice!
